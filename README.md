# curso-node-tablas
###Opciones:
```

      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     el valor base de la tabla                  [número] [requerido]
  -l, --listar   para ver en consola la tabla creada [booleano] [defecto: false]
  -h, --hasta    el valor hasta el cual va mostrarse la tabla
                                                          [número] [defecto: 10]

```
